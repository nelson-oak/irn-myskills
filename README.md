## Iniciando o projeto

Após clonar o projeto, é necessário atualizar as dependências.

### Comandos para baixar dependências e iniciar a aplicação

```bash
yarn
yarn start
```

### Executar no android ou no iOS

```bash
yarn android
yarn ios
```
